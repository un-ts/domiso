## Usage

### Install

```sh
# pnpm
pnpm add domiso

# yarn
yarn add domiso

# npm
npm i domiso
```

### API

```js
import { IMAGE_SVG_XML, sanitize, sanitizeSvg } from 'domiso'

sanitize(`<!-- HTML content -->`)
sanitize(`<svg>SVG content</svg>`, IMAGE_SVG_XML)
sanitizeSvg(`<svg>SVG content</svg>`)
```
